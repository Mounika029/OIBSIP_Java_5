# Library Management System

## 📌 Objective
To develop a Java-based Library Management System that helps manage books, users, and issue/return operations efficiently.

## 🛠 Tools & Technologies
- Java
- OOP Concepts
- Collections (ArrayList / HashMap)
- Scanner (for user input)
- (Optional) Swing for GUI

## 🚀 Features
- Add new books
- View available books
- Issue books to users
- Return books
- Maintain book records
- Simple user interaction menu

## 🧩 Steps Performed
1. Designed book and user classes
2. Created data structures to store book details
3. Implemented add and view book features
4. Developed issue and return functions
5. Built menu-driven program for user interaction

## 📊 Outcome
The Library Management System works successfully, allowing efficient management of books and user transactions.

## ▶️ How to Run
1. Open the project in your IDE (VS Code / IntelliJ)
2. Compile the Java files
3. Run the main class
4. Use menu options to perform operations
